# Kitty Date Invitation

一个可以直接用 GitHub Pages 发布的静态约会邀请网页。

发布方式：

1. 在 GitHub 新建一个公开仓库。
2. 上传本目录下的 `index.html`、`styles.css`、`script.js`、`.nojekyll` 和 `assets/date-bg.png`。
3. 进入仓库 `Settings -> Pages`。
4. Source 选择 `Deploy from a branch`，Branch 选择 `main` 和 `/root`。
5. 保存后等待 1-3 分钟，即可得到 GitHub Pages 链接。
